About


This is a simple socket application that lets you connect and control your IOT Boards such as ESP8266, ESP32, Raspberry Pi series, and more, from anywhere without manually configuring your Wi-Fi router to be publically available, Just by using the telegram application. This Application Have both Free and paid socket Connectivity and also let your boards connect with Websocket and TCP Socket.
NOTE: In case of any Service misuse, your telegram account can be permanently banned from using our services.

Connection


Admin :

  • Sign in to elsocket with Google Account
  • Open telegram bot
  • Paste your Admin token (From Dashboard) into the bot to connect
  • Add the Client token into your IOT board


Devices :


TCP :
  • Connect your IOT Board to 148.113.3.57 at port 3000
  • Send Any Client token (From Dashboard) to server to verify device
WEBSOCKET :
  • Connect your IOT Board to 148.113.3.57 at port 8080
  • Send Any Client token (From Dashboard) to server to verify device

Usage


Bot Commands :
  • /status : Get current Connection Status
  • /devices : List all Available Devices
  • /selected : Get Selected Device
  • /stop : Disconnect Current session
